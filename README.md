# 🎓 Student Management System

A Django-based Student Management System with CRUD operations.

## Features

- Create Student
- Read Student
- Update Student
- Delete Student
- SQLite Database
- Django Class Based Views

## Tech Stack

- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite

## Installation

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
